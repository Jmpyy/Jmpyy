Actualmente curso mis estudios en la **Universidad Provincial de Ezeiza (UPE)** y me especializo en la integración de tecnologías eficientes para resolver problemas complejos, abarcando desde la arquitectura de bases de datos hasta la interfaz de usuario.

*   🔭 **Actualidad**: Desarrollando plataformas logísticas y sistemas de administración de flotas.
*   🌱 **Aprendizaje**: Profundizando en **Arquitectura de Software**, **Seguridad Backend** y lenguajes de bajo nivel como **C**.

### 📦 Proyectos Destacados

#### [🏥 Integrar Salud](https://github.com/Jmpyy/Integrar-salud)
*Sistema de Gestión Clínica (SaaS) integral para la salud mental y consultorios médicos.*
*   **Tecnologías**: React 19, Vite, Zustand, Tailwind CSS, Recharts.
*   **Descripción**: Plataforma completa que centraliza agendas, facturación inteligente multimoneda y seguimiento de historias clínicas. Implementa un dashboard administrativo avanzado con analíticas financieras en tiempo real.

#### [🚚 Rutabit](https://github.com/Jmpyy/Rutabit-landing-page)
*Ecosistema de logística y seguimiento vehicular inteligente.*
*   **Tecnologías**: HTML5, CSS3 Moderno, JavaScript (ES6+).
*   **Descripción**: Landing page corporativa y sistema de gestión optimizado para el seguimiento de activos logísticos, con un fuerte enfoque en UI/UX moderna y rendimiento de carga.

#### [🩸 Sistema de Donación de Órganos (EDA)](https://github.com/Jmpyy/Proyecto-EDA1)
*Software de bajo nivel para la gestión crítica de trasplantes y donaciones.*
*   **Tecnologías**: C, OpenSSL (SHA-256), Archivos Binarios, Heaps de Prioridad.
*   **Descripción**: Implementación robusta de estructuras de datos dinámicas para la asignación de órganos según niveles de urgencia médica. Cuenta con encriptación de seguridad para datos sensibles y persistencia eficiente.

#### [🌍 Traccar-web (Fork)](https://github.com/Jmpyy/rutabit-monitoreo)
*Personalización de plataforma GPS abierta.*
*   **Tecnologías**: React, JavaScript Core.
*   **Descripción**: Adaptación y mejora de módulos frontend para la visualización de geolocalización en tiempo real y gestión de flotas masivas.
